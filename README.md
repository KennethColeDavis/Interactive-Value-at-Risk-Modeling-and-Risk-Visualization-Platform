# Interactive-Value-at-Risk-Modeling-and-Risk-Visualization-Platform
Interactive financial risk analytics platform that models portfolio Value-at-Risk (VaR) using historical, parametric, and Monte Carlo methods with a Python backend and web dashboard.


Install Instructions
====================

Clone the repo:
- Navigate to desired file location of your project
- Open in terminal and type `git clone https://github.com/KennethColeDavis/Interactive-Value-at-Risk-Modeling-and-Risk-Visualization-Platform.git`
- This will clone the repo of to your local machine.

Set up environment and downlaod requirements:

- Download and install the latest version of python
- Cd into the folder and type ` python3 -m venv CS498 && source CS498/bin/activate`
- This will create and activate a virtaul environment to contain all dependiences
- At any time, you can type `source CS498/bin/activate` to activate the environment or type `deactivate` to deactivate the environment.
- We can now install all dependiences by typing
 - `pip install -U pip`
 - `pip install -r requirements.txt`
